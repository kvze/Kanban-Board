## Kanban Board

Небольшое веб‑приложение для управления задачами в формате Kanban‑доски.  
Позволяет добавлять задачи, переносить их между колонками и отмечать выполнение.

## Демо

Приложение доступно по ссылке:  
[`https://kanbanboard-kvze.netlify.app/`](https://kanbanboard-kvze.netlify.app/)

## Технологии

- **HTML**, **CSS**
- **Vanilla JavaScript (ES6+)**
- Работа с DOM и `localStorage` для сохранения задач

После запуска вы увидите доску с колонками **In queue**, **In progress** и **Done**, куда можно добавлять и перетаскивать задачи.
